Current behavior Auto = `azColsForWidth(width)`，只看畫布寬度；2/3/4 是手動欄數鎖定；欄寬等分。
Proposed Auto Auto = session-aware grid：先用寬度算上限，再用 session 數量選 1/2/3/4 欄，避免少量 session 被切太碎。
Drag gutter control 當 resolved columns = 2 時顯示中間 gutter，像 sidebar 一樣拖曳調整比例；其他欄數仍等分且不顯示 handle。

判讀與假設

1`AgentZone.tsx` 目前用 `resolveAzCols(azColsOverride, azWidth)` 產生欄數，然後直接 `repeat(cols, minmax(0, 1fr))`。
2`v2-store.ts` 目前只有 `azWidth` 與 `azColsOverride`，沒有左右比例狀態；要落地需要新增 `azSplitRatio` 或類似欄位。
3「左右大小」此稿定義為兩欄 grid 中間的 draggable resizer；這可同時支援 Auto resolved 2 columns 與手動 Lock 2。
